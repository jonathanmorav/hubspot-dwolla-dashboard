üêõ Common Issues & Solutions

Issue: "Cannot read properties of undefined" error
Solution: The service worker might not be active. Check the service worker console and reload the extension.
Issue: OAuth popup doesn't open
Solution: Check if popup blocker is enabled. Also verify the Extension ID in backend/.env matches your loaded extension.
Issue: "Client ID not configured" error
Solution: Environment variables might not be loaded. Check if .env file exists and rebuild the extension.
Issue: Network errors or CORS issues
Solution: Ensure backend server is running on port 3001. Check backend console for errors.
